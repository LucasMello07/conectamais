# 🔗 Conecta+

Plataforma web para **conectar usuários a profissionais autônomos**, facilitando a busca, comunicação e contratação de serviços de forma simples e intuitiva.

## 🚀 Principais Pontos

* **Problema:**
  Dificuldade no acesso a profissionais autônomos e baixa visibilidade dos serviços no meio digital

* **Solução:**
  Plataforma centralizada com busca, perfis detalhados, avaliações e interação direta entre usuários e prestadores

* **Funcionalidades-chave:**
  Autenticação de usuários, ranking de profissionais, gestão de perfis e navegação intuitiva
